# 📈 Advertising Sales Prediction Dashboard

## Overview

The Advertising Sales Prediction Dashboard is a Machine Learning-powered web application developed using Streamlit. It helps businesses analyze advertising expenditures across different marketing channels and predict future sales using regression techniques.

The dashboard provides interactive visualizations, performance metrics, sales forecasting, and actionable business insights to support data-driven marketing decisions.

---

## Features

### 📊Data Analysis

* Upload Advertising dataset (.csv)
* Dataset preview and summary
* Missing value and duplicate data analysis
* Statistical overview of advertising spend and sales

### 📈Interactive Visualizations

* TV Advertising vs Sales
* Radio Advertising vs Sales
* Newspaper Advertising vs Sales
* Correlation Heatmap
* Actual vs Predicted Sales Comparison

### 🤖Machine Learning Model

* Linear Regression Model
* Train-Test Split Evaluation
* Model Performance Metrics:

  * R² Score
  * Mean Absolute Error (MAE)
  * Root Mean Squared Error (RMSE)

### 🔮Sales Forecasting

Predict future sales based on:

* TV Advertising Budget
* Radio Advertising Budget
* Newspaper Advertising Budget

### 💡Business Insights

* Advertising channel impact analysis
* Marketing effectiveness evaluation
* Budget optimization recommendations
* ROI improvement strategies

## 👩‍💻Installation

### Clone Repository

```bash
git clone https://github.com/your-username/advertising-sales-prediction.git
cd advertising-sales-prediction
```

### Install Dependencies

```bash
pip install streamlit pandas numpy scikit-learn plotly statsmodels
```

---

## Run Application

```bash
streamlit run app.py
```
